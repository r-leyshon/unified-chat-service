# Sample app documentation

User guidance for three simple example apps. Each subfolder matches a **project** you can create in the Document Library; upload these files to test RAG and the chat assistant.

| Project / app      | File                | Format   | Style          |
|--------------------|---------------------|----------|----------------|
| Tip Calculator     | `tip-calculator/user-guide.md`     | Markdown | Step-by-step guide |
| Unit Converter     | `unit-converter/reference.txt`     | Plain text | Reference (units, limits) |
| Minimal Pomodoro   | `minimal-pomodoro/getting-started.md` | Markdown | Getting started + FAQ |

Upload the whole folder contents (or individual files) per project in [/library](http://localhost:3000/library), then ask the chat questions about each app.
